🚴 QB-C01S Web Bluetooth 接続テスト

まずは 接続できるか と どのBluetoothサービスが見えるか を確認するためのテストページです。
速度やケイデンスが取れれば、あとで「一定速度未満でYouTubeを停止」のロジックに発展できます。

✅ 使い方

  • Chrome / Edge 系ブラウザで開いてください（iPhone Safari は不可のことが多いです）。
  • HTTPS か localhost で開いてください。単なる file:// では Web Bluetooth が使えません。
  • まずバイクの電源を入れて、他のアプリ（NEXGIM、Zwift、MyWhooshなど）が接続していない状態にしてください。
  • うまくいかないときは、最初は「名前指定で接続」、見つからなければ「全デバイスから選ぶ」を試してください。
